# Organizador de Declaração de Imposto de Renda em Excel

## Descrição do Projeto

Este projeto foi desenvolvido como parte de um desafio da plataforma de ensino **DIO (Digital Innovation One)** por um estudante iniciante da área de tecnologia e dados. O objetivo principal foi aplicar, na prática, conceitos básicos de Excel e organização de dados em um cenário próximo ao real.

O desenvolvimento do projeto seguiu as etapas abaixo:

1. Análise do problema de organização das informações necessárias para a declaração de imposto de renda;
2. Definição do Excel como ferramenta para centralizar e estruturar os dados;
3. Criação de uma planilha com foco em organização e clareza das informações;
4. Implementação de validações simples para melhorar a qualidade dos dados inseridos;
5. Estruturação da navegação para facilitar o uso da planilha.

A planilha funciona como um **agregador de dados**, permitindo reunir informações em um único local, seguindo uma lógica simples de organização, comum em projetos iniciais da área de dados.

## Objetivo do Desafio

O objetivo deste desafio foi desenvolver uma ferramenta em Excel capaz de:

1. Organizar dados básicos utilizados na declaração de imposto de renda;
2. Facilitar o controle e a visualização das informações;
3. Minimizar erros de preenchimento por meio de validações;
4. Aplicar conceitos iniciais de organização e tratamento de dados.

Todo o projeto foi desenvolvido utilizando recursos básicos do Excel, com foco no aprendizado prático.

## Funcionalidades

1. Registro estruturado de informações relacionadas ao imposto de renda;
2. Validação de dados para garantir maior consistência das informações;
3. Menus e links simples para navegação entre as abas;
4. Organização dos dados em abas separadas;
5. Interface simples e fácil de utilizar.

## Ferramentas Utilizadas

* Microsoft Excel

  1. Fórmulas básicas
  2. Validação de dados
  3. Organização e estruturação de planilhas

## Objetivos de Aprendizagem

Com a realização deste desafio, foi possível:

1. Aplicar conceitos iniciais de tecnologia e dados utilizando Excel;
2. Entender melhor a importância da organização e padronização de dados;
3. Aprender a utilizar validações para melhorar a qualidade dos dados;
4. Praticar a documentação de um projeto técnico de forma simples e objetiva.

## Estrutura do Projeto

O projeto é composto por um único arquivo Excel que contém:

1. Abas organizadas por tipo de informação;
2. Campos preparados para entrada e validação dos dados;
3. Elementos simples de navegação entre as seções.

## Como Utilizar

1. Faça o download do arquivo Excel disponível neste repositório;
2. Abra o arquivo no Microsoft Excel;
3. Navegue pelas abas da planilha;
4. Preencha os campos conforme solicitado;
5. Utilize os menus e links para acessar as diferentes seções.

## Observações

1. Este projeto possui finalidade educacional;
2. A planilha tem como objetivo apoiar a organização e o controle de dados;
3. Para a declaração oficial do imposto de renda, recomenda-se a orientação de um profissional especializado.
